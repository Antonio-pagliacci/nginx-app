Docker

Docker é uma plataforma para criar, empacotar, distribuir e executar aplicações em containers. Ele permite que desenvolvedores criem ambientes isolados e portáveis que funcionam da mesma forma em qualquer lugar (ambiente de desenvolvimento, teste, produção etc.).

Conceitos Principais

Container: É uma unidade leve, portátil e isolada que roda uma aplicação com todas as suas dependências. Imagem: É um blueprint (modelo) imutável usado para criar containers. Pode ser baseada em sistemas operacionais como Ubuntu, Alpine etc. Dockerfile: Arquivo de texto com instruções para construir uma imagem. Docker Hub: Repositório online de imagens Docker (como um "GitHub de imagens"). Volume: Usado para persistir dados fora do container. Docker Compose: Ferramenta para definir e gerenciar múltiplos containers (como uma stack) usando um arquivo docker-compose.yml.

Vantagens do Docker

Portabilidade entre ambientes Isolamento de processos Rápida criação e destruição de ambientes Reprodutibilidade (mesmo comportamento em qualquer lugar)
